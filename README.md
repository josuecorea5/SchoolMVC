# School System.
### This project handles information related to school system's processes such as:
- Students.
- Grades.
- Study plans.
- Teachers.
- Subjects.
- Enrollments.
- Users.

### Technologies:
- .NET 8.
- Entity Framework.
- MVC.
- Razor pages.
- JavaScript.
- SQL Server.
- Cloudinary.
- Highcharts.
- Identity.
